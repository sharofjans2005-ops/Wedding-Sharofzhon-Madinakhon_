## 📥 Как загрузить музыку?

### Шаг 1: Выберите сайт (рекомендуем Pixabay)
1. Перейдите на https://pixabay.com/music/
2. Введите поисковый запрос: "elegant wedding" или "romantic instrumental"

### Шаг 2: Найдите понравившуюся музыку
3. Кликните на трек
4. Послушайте предпросмотр
5. Нажмите кнопку "Download"

### Шаг 3: Сохраните файл
6. Выберите качество (высочайшее качество)
7. Файл скачается в формате MP3

### Шаг 4: Добавьте на сайт
7. Откройте папку `music/`
8. Поместите файл `wedding-music.mp3` туда
9. Готово! Музыка будет играть на сайте

## ⚠️ Важные замечания

1. **Проверьте лицензию:** Убедитесь, что музыка свободна для использования
2. **Качество файла:** Используйте MP3 со скоростью битрейта 128 кбит/с или выше
3. **Длина файла:** Не делайте файл слишком длинным (зацикливается)
4. **Размер файла:** Для быстрой загрузки сайта используйте файлы до 5 МБ

## 🔊 Установка громкости

В файле `script.js` найдите строку:
```javascript
backgroundMusic.volume = 0.3; // Установить громкость на 30%
```

Измените значение (0 до 1):
- 0.1 = очень тихо
- 0.3 = тихо (текущее значение)
- 0.5 = средняя громкость
- 1.0 = максимальная громкость
