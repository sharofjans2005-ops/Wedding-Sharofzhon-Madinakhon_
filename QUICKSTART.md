## 🔧 Решение проблем

### Проблема: Музыка не воспроизводится
**Решение:**
1. Проверьте, что файл находится в `music/wedding-music.mp3`
2. Убедитесь, что это MP3 файл
3. Откройте консоль браузера (F12) и посмотрите на ошибки
4. Попробуйте другой браузер

### Проблема: Сайт не открывается
**Решение:**
1. Убедитесь, что файл `index.html` находится в правильной папке
2. Используйте локальный веб-сервер вместо прямого открытия
3. Проверьте, что имя файла ровно `index.html` (без пробелов)

### Проблема: Текст на русском отображается неправильно
**Решение:**
1. Браузер должен поддерживать UTF-8
2. Попробуйте другой браузер (Chrome, Firefox)
3. Очистите кэш браузера

### Проблема: Карта не отображается
**Решение:**
1. Это нормально - требуется API ключ от Яндекса
2. Сайт полностью работает без карты
3. Кнопка "Построить маршрут" все равно работает

## 📱 Тестирование на мобильном

### Через USB:
1. Подключите телефон к компьютеру
2. Запустите веб-сервер: `python -m http.server 8000`
3. Найдите IP адрес компьютера: `ipconfig` в Command Prompt
4. На телефоне откройте: `http://<IP>:8000`

### Через облако:
1. Загрузьте папку на веб-хостинг
2. Откройте ссылку на телефоне

## 🎨 Кастомизация

### Изменить цвет темы:
1. Откройте `styles.css`
2. Найдите `#ff69b4` (розовый цвет)
3. Замените на нужный (например: `#ff1493`)

### Изменить текст:
1. Откройте `index.html`
2. Найдите текст в `<p>` тегах
3. Отредактируйте

### Изменить шрифт:
1. Откройте `styles.css`
2. Найдите `font-family: 'Georgia'`
3. Измените на нужный шрифт
