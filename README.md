## 🚀 Быстрый старт

### 1️⃣ Локальное тестирование

#### На Windows (PowerShell):
```powershell
cd c:\Users\sarof\Desktop\свабьда
python -m http.server 8000
```

Затем откройте в браузере: **http://localhost:8000**

#### Или просто дважды кликните на `index.html`

### 2️⃣ Добавление музыки

1. Скачайте MP3 файл (рекомендуется с Pixabay.com)
2. Поместите его в папку `assets/music/`
3. **Переименуйте на:** `song.mp3`

Готово! Музыка будет автоматически воспроизводиться.

### 3️⃣ Публикация на GitHub Pages

#### Способ A: Через GitHub Web Interface (самый простой)

1. Создайте репозиторий на GitHub с именем `username.github.io`
2. Загрузьте все файлы в корень репозитория
3. Сайт будет доступен по адресу: `https://username.github.io`

#### Способ B: Через Git (командная строка)

```bash
# Инициализируем Git
git init

# Добавляем все файлы
git add .

# Коммитим
git commit -m "Initial commit: wedding website"

# Добавляем удаленный репозиторий
git remote add origin https://github.com/your-username/your-repo-name.git

# Отправляем на GitHub
git push -u origin main
```

Откройте `style.css` и найдите переменные в `:root`:
```css

--secondary-gold: #f4d03f;    /* Вторичный золотой */
--cream: #faf8f3;             /* Кремовый */
```

### Изменить шрифты
Все шрифты подключены из Google Fonts:

- **Основной текст:** `Montserrat`

### Изменить дату свадьбы
В `script.js` найдите:
```javascript
const weddingDate = new Date('2026-07-11T18:00:00').getTime();
```

### Изменить место
В `script.js` найдите функцию `initializeMap()`:
```javascript

```

В `index.html` найдите:

## 🐛 Решение проблем

### Проблема: Музыка не воспроизводится
**Решение:**
1. Проверьте, что файл находится в `assets/music/song.mp3`
2. Убедитесь, что это MP3 файл
3. Откройте консоль браузера (F12) и посмотрите ошибки

### Проблема: Карта не отображается на GitHub Pages
**Решение:** API ключ предоставлен в коде. Это нормально.

### Проблема: Текст на русском отображается неправильно
**Решение:** Браузер должен поддерживать UTF-8 (все современные браузеры это делают).

### Проблема: Сайт не загружается на GitHub Pages
**Решение:**
1. Убедитесь, что репозиторий публичный
2. Проверьте имя репозитория (должно быть `username.github.io` или параметр Pages)
3. Подождите несколько минут после загрузки файлов
